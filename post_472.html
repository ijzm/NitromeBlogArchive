Rubble Trouble UpdateWed, 20 Jul 2011

Hi Nitromians!

Rubble Trouble is now... Rubble Trouble New York!

With the soon to come Rubble Trouble Tokyo, we wanted to give the original game a set location and set the series up so that more locations might become themes for possible future releases after Rubble Trouble Tokyo.

So what does this mean other than a name change? Well, there is an updated background and title screen but the main draw comes from the addition of two brand new levels with the new last level in particular providing an interesting surprise that is very appropriate to New York! :)
