Accounts Coming Next Week!Fri, 05 Jul 2013

Guess what Nitromians...

We're launching accounts next week! Hurray!

Check out Austin in this Video using Nitrome accounts.

That's right, we are just about ready to start rolling out the accounts system for you all. We know it has been a loooong time coming (It was a pretty deep system update for us). Being as thorough as we have been should allow us to start introducing more features down the line more quickly.

As we spoke about this in previous account related articles, this first revision will be the foundations upon which we will build.

One of the things you'll notice we focused on in the video is the ability to find Avatars in games. We have been getting a lot of enjoyment from adding these in to our games, we hope it translates into the experience you'll have when trying to find them all!

Again, we should also stress here that we really want to iron out any of those early bugs, that you inevitably find in any system this size, before we start loading down the servers with even heavier features. Game Saving for instance. No one wants to lose their game saves. Rest assured that we have already looked into this to find the best ways to handle saves ;)
