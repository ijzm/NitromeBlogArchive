Hearting SystemWed, 30 Mar 2011

The more observant amongst you may have noticed a heart symbol in one of our designs for the new site and today we want to let you know a little more about what the intent is with this element. It is in essence a ‘like’ button so if you click it in the new system it means you like it or as we will probably be calling it ‘heart’ it or possibly ‘love’ it…..but that might be a bit too girly for the boys

What is probably more interesting is how we are going to use that information. For starters you will be able to see the games you have hearted in a section within ‘games’ so there will be a quick easy way to find your favorites. You will also be able to see with a quick rollover on the front page which games are hearted there and we will in time add more ways to view your hearted games more easily

This feature won’t only be used for you however…..we will be using the number of ‘hearts’ a game gets to gauge popularity and the first place you will see the effect of this will be in the top 5 games which in the new design will be based on the (daily) number of ‘hearts’ the games gets. There will also likely be an area in games featuring the games in popularity order!

In the future the hearting system will apply to other areas of the site too so you will be able to rate other things and interact in other ways that we are not ready to mention.

Stay tuned for more updates!
