Hi Nitromians!

The website is currently undergoing some major changes in both our frontend and backend systems as from our last blog update and today I thought we'd give you a quick progress update as to where we are with the redesign.

The website has been built, and now needs a few tweaks here and there. We have also been making room for even more functionality that we plan to add to the website after the initial launch.

I have got the 'hearting' system working for the game pages which means you can help shape which of our games make it into the top five each day!

A backend system has also been created to help for making a website skin quicker, giving the artist more time to design those much loved skins and backgrounds!

We’ve got the ‘new games’ indicators working too, these will be used to show you which games you haven’t played yet!

There's still a little bit more work to do including ironing out some browser bugs and updating the way we launch and update games. At the moment I am still coming across bugs that need tweaking regularly.

I'm also updating the website's scroll functionality to make it more friendly to navigate!

We will bring you more soon!

by Tom
