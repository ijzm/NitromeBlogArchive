Steamlands Update!Fri, 05 Aug 2011

Hi Nitromians,

Our Steamlands game just got a great update. There is now a dev mode in the game which you can access from the title screen.

This will allow you to change the game in pretty much any way you see fit. You're able to make new tanks, new levels or even a whole new map if you have the time and persistence!

For those who do make great levels we would love to know so we are inviting people to send them in to fans@nitrome.com. The best levels will be added to the game in the game’s first update and if you give your name in the email we will make sure your credited for the mission :)

At the moment we have no end date for submissions as we want to see how many we get sent before deciding any cut off but if you want the best chance of getting your level/levels in the game then sooner is going to be better than later!
