Demos Coming Soon!Fri, 16 Mar 2012

We have got a slightly different Friday Update for you this week!

We’re thrilled to announce the start of a new feature coming soon to Nitrome.com

We are going to start offering demos for selected upcoming games. The benefits of doing this is to gain useful feedback before progressing too far with the game and to show you guys what we are currently up to.

Please note that we are not going to do demos for every game. We feel that for some games it would spoil the enjoyment of exploring a new mechanic for the first time. But, for the right games, it lets you try what we’re up to and help shape it into a better game before the final game launches.

So what’s a demo going to be? Is it a tech demo? Is it a small portion of the final game? Yes, no, well maybe… The state that the demo takes will vary on a game by game basis. For the first demo, it is more of a small playground where you will get to see some of the mechanics that we are developing. It contains a lot of final art, but some art is not there yet so the final product will look a lot better still despite it being presentable at this stage. Even though it’s not a final game, and not a level from the final game, the demo has a basic challenge in place so there is a point to play it other than finding out what we are up to.

So what is the demo and when is it going to be out? …erm, the demo will be out pretty soon, but we don’t want to give a date that we might not stick to. What we can say is that the first demo is based on one of our existing franchises, but not one that we have currently announced (so not the upcoming Cave Chaos or Hot Air follow ups).

We hope you enjoy the rest of your Friday and have a great Weekend!
